Работа с приложением:
 Как открыли приложение, нужно сделать полноэкранный режим.

 При нажатии клавиши "P" вызызывается окно настроек, где можно изменять характеристики объекта.
 Характеристики:
	Size - размер
	Vx - скорость по оси x
	Vy - скорость по оси y
	Disappear(Dis) - пропадание. Вводится значение в миллисекундах!!!(1000 мс = 1с). По умолчанию стоит 0
	SpeedWrite - скорость записи(принимает значения от 0 до 5 в соответствии в режимами айтрекера)
	Flight Mode - режим полёта. Принимает 0 и 1(по умолчанию 0), где 0 - шарик отскакивает от стенок и 1 - при касании со стеной, шарик появляется с противоположной стороны.
	Tail - шлейф. Принимает 0 и 1(по умолчанию 0). При 0 шлейфа нет, при 1 - есть.

 Когда все нужны настройки были введены, нажимаете на крестик в верхнем правом углу этого экрана.
 
 Также в главном окне есть возможность динамически менять характеристики:
	"+" - увеличивает размер;
	"-" - уменьшает размер;
	"Up" - увеличивает скорость по Y;
	"Down" - уменьшает скорость по Y;
	"Right" - увеличивает скорость по X;
	"Left" - уменьшает скорость по X;

Запись ведётся в файле с расширением .csv. Он появится в папке, где хранится .exe файл.
